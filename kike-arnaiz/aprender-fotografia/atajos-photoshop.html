Comandos de Photoshop + Descargables

Los programas de edición fotográfica como Photoshop, tienen un montón de menús, opciones y funciones para sacarle provecho, sin embargo, en ocasiones se vuelve una tarea cansada tener que ir clicando sobre cada menú para activar cierta función.

Gracias a los comandos o atajos de Photoshop puedes controlar o activar las funciones para ahorrar tiempo al momento de editar.

En este artículo he preparado una lista con algunos de los comandos que debes de conocer de Photoshop, los cuales te ayudarán a hacer que tu experiencia al editar sea más amena. 

Si no puedes comprar Photoshop mira este artículo, en el que te muestro diferentes programas gratuitos para editar tus fotos.

Ahora sí, veamos cuáles son los atajos o comandos de Photoshop que debes de conocer, no importa si usas Windows o Mac, te mostraré ambas versiones.

Vale decir que cuando hablemos de Ctrl en Windows, el equivalente en Mac será CMD y si nos referimos a la tecla Alt, en Mac sería el OPT.

Comandos de Photoshop Basicos 

Existen atajos para diferentes situaciones, sin embargo, hay atajos básicos que todo fotógrafo debería dominar en Photoshop, aquí te dejo una lista con algunos de estos.

  • Ctrl + N, sirve para crear un nuevo documento.

  • Ctrl + S, para guardar el archivo sobre el cual estás trabajando, aunque yo suelo utilizar con mayor frecuencia el siguiente comando.

  • Mayús + Ctrl + S, que es el atajo para guardar como…allí elijo como quiero guardar el archivo sobre el cual estoy trabajando y no daño la fotografía original. 

  • Ctrl + Alt + Mayús + S, es para exportar archivos para web, genial para guardar fotografías listas para subir a redes sociales o sitios web.

  • Ctrl + +, para acercar la imagen.

  • Ctrl + -, para alejar la imagen.

  • Ctrl + T, transformación libre de imagen.

  • Ctrl+J, crear capa duplicada.

  • X, para cambiar de color de forma automatica entre los dos que tienes seleccionados en la paleta. Esta función viene genial para cuando estás trabajando con máscaras de capas.

  • Ctrl+C, para copiar.

  • Ctrl+V, para pegar.

  • Ctrl+Z, para deshacer.

  • Ctrl+Mayús+Z para rehacer.

  • Ctrl+x para cortar.

Estos a mi parecer son algunos de los básicos que debes de conocer y dominar, para comenzar a editar tus fotografías. Pero si quieres saber cuales son todos los comandos, continúa leyendo:

Activa las herramientas con atajos

El panel de herramientas es bastante extenso, por lo que siempre viene bien conocer cuales son los comandos para activar cada una de estas herramientas y mejorar tu fluidez al momento de hacer tus ediciones.

Si dejas el cursor sobre la herramienta te dirá el nombre de la misma y el comando, pero aquí te dejo una lista con todos los atajos para herramientas.

  • V: Herramienta Mover o Mesa de trabajo

  • M: Marco rectangular o elíptico

  • L: Herramienta de lazo 

  • W: Herramienta de selección rápida o varita mágica

  • C: Herramienta recortar.

  • I: Cuentagotas

  • J: Corrector puntual, si le das mayús cambiará entre diferentes herramientas como pincel corrector, herramienta parche, movimiento con detección de contenido y pincel de ojos rojos.

  • B: Pincel

  • S: Tampón clonar

  • Y: Pincel de historia

  • E: Borrador

  • G: Degradado

  • O: Sobrexponer

  • T: Herramienta de texto

  • P: Pluma 

  • A: Selección de trazado 

  • H: Herramienta mano

  • Z: Zoom

  • X: cambiar entre color frontal y fondo

Atajos de Photoshop para edición de imágenes

Si te dedicas a la fotografía si o si vas a utilizar herramientas para la postproducción de imágenes como los tonos, saturación, niveles, curvas, exposición. Sin embargo, utilizarlas de la forma clásica puede no ser la mejor opción si queremos ser productivos a la hora de editar. 

Así que por ello te dejo una lista con los comandos que debes de conocer para retocar tus fotos y ahorrarte unos cuantos segundos entrando al menú de Imagen > Edición y, por último, seleccionar la opción que querías.

  • Ctrl + L, con esta accedes de forma directa al ajuste de Niveles.

  • Ctrl + M, permite acceder al ajuste de Curvas.

  • Ctrl + U, Tono/Saturación.

  • Ctrl + B, entra directo a la opción Equilibrio de color.

  • Ctrl+ Alt + Mayús + B, para convertir la fotografía en Blanco y negro.

  • Ctrl + I, para Invertir los colores de la imagen sobre la cual estás trabajando.

  • Ctrl + Mayús + U, accede a la opción Desaturar, que le quita los colores a la foto.

  • Ctrl + Mayús + L, Tono automático.

  • Ctrl + Alt + Mayús + L, Contraste automático.

  • Ctrl + Mayús + B, Color automático.

  • Alt + Ctrl + C: Tamaño de lienzo.

  • Alt + Ctrl + I, Tamaño de imagen.

Comandos para seleccionar en Photoshop

Una de las tareas que hacemos en Photoshop cuando vamos a editar nuestras fotografías o hacer diseños es seleccionar objetos. Por ello, es importante utilizar comandos para optimizar el tiempo y la forma de ejecutar una tarea simple como hacer una selección.

  • Ctrl + A: Seleccionar todo.

  • Ctrl + D: Deseleccionar todo.

  • Ctrl + Mayús + I: Invertir la selección.

  • Ctrl + Mayús + Alt + N: Crear una nueva capa a partir de la selección.

  • Ctrl + J: Copiar la selección en una nueva capa.

  • Ctrl + Mayús + J: Cortar la selección en una nueva capa.

  • Ctrl + Alt + J: Copiar la selección en una nueva capa sin borrar la selección original.

  • Ctrl + Mayús + Alt + J: Cortar la selección en una nueva capa sin borrar la selección original.

  • Ctrl + Mayús + Alt + E: Copiar la selección en una nueva capa fusionada.

  • Ctrl + Mayús + Alt + C: Copiar la selección visible en una nueva capa.

  • Ctrl + Mayús + Alt + V: Pegar la selección en una nueva capa.

¿Cómo personalizar un atajo en Photoshop?

Para conocer todos los atajos que hay disponibles en Photoshop puedes utilizar el comando Alt+Mayús+ctrl+k, aunque para mi es más cómodo activar el comando en este orden: Ctrl+Mayús+Alt+k (funciona igual).

Al activar la ventana, te aparecerán ordenados por categorías e incluso, si hay alguna función a la cual no puedas acceder por medio de comandos, tú puedes asignarle un comando personalizado.

¿Cómo se hace?

Selecciona la sección en la que se encuentra la acción a la que quieres crearle un comando, por ejemplo:

En mi caso quiero cambiar la forma en que hago el Guardar como…

Así que me voy a la sección Archivo, clico en la flecha que está antes de la sección y allí se despliegan todas las opciones que aparecen en ese menú. 

Ahora me voy a la opción Guardar como y clico sobre el comando que aparece a la derecha, allí podré agregar el comando que yo quiera, por ejemplo, Ctrl+Mayús+G.
